   - **Opera**: `opera://extensions/`

## üöÄ Usage

1. Open [winna.com](https://winna.com) and log into your account

2. Click on the extension icon in the browser toolbar

3. Enable auto-join by clicking the big toggle button

4. Set your desired delay (in seconds):
   - `0` - instant join
   - `3` - recommended (default)
   - `10+` - if you want to wait before joining

5. Keep the winna.com tab open
